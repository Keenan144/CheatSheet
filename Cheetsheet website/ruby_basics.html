Ruby Basics

puts:
prints following info, and spaces it out.

gets:
takes user info from command line as a program runs

each_statement {|| .... }
Enumerate each statement constructed from the defined vocabulary terms.

do
makes a line of code run

loop
repeat line of code until prompted otherwise

end
end a program

if/ else
run a line of code if a condition exists, else run the other code.

while
run this line of code, while this condition exists

case
when a condition exists, do something. End no loop.

class
holds methods inside. use classes to clean up code

methods (def)
line of code, use methods to make classes

.to_s
change an object to a string

.to_i
change an object to an integer

.upcase/.capitalize/.downcase
make a string upcased, capped or downcase.
